📖 일본어 3가지 문자

일본어는 크게 3가지 문자 체계를 사용합니다. 한국어는 한글만 있지만, 일본어는 상황에 따라 다르게 쓰여요.

1️⃣ 히라가나 (ひらがな)

  • 부드러운 곡선의 글씨
  • 일상 일본어에서 가장 자주 사용
  • 기본 44자 (+ 탁음/반탁음)

2️⃣ 카타카나 (カタカナ)

  • 각진 직선의 글씨
  • 외래어, 의성어에 사용
  • 기본 44자 (+ 탁음/반탁음)

3️⃣ 한자 (かんじ)

  • 중국에서 유래한 문자
  • 뜻과 음을 나타냄

💡 팁: 지금은 히라가나와 카타카나만 집중하세요! 한자는 나중에 배워도 괜찮아요.

📝 ひらがな 기본 44자 (1)

あいうえお

かきくけこ

さしすせそ

たちつてと

なにぬねの

🔊 발음 팁

  • あ (a) - "아" / い (i) - "이" / う (u) - "우"
  • え (e) - "에" / お (o) - "오"

💡 5개의 모음(あ, い, う, え, お)을 먼저 완벽하게 익히면, 나머지 글자들은 자음 + 모음 조합이므로 쉬워집니다!

📝 ひらがな 기본 44자 (2)

はひふへほ

まみむめも

やゆよ

らりるれろ

わを (ん)

⚠️ 주의점

  • ん (n) - 유일한 모음 없는 글자
  • わ (wa) - 현대 일본어에서 거의 미사용
  • を (wo) - 목적어 조사로만 사용

💡 팁: わ행과 を는 거의 실생활에서 안 나오니 나중에 복습해도 괜찮아요!

📝 カタカナ 기본 44자 (1)

アイウエオ

カキクケコ

サシスセソ

タチツテト

ナニヌネノ

🔊 발음 팁

카타카나의 발음은 히라가나와 동일합니다! 단지 글씨 모양만 다릅니다.

카타카나 vs 히라가나

あ (히라가나) = ア (카타카나) → 둘 다 "a" 발음

📝 カタカナ 기본 44자 (2)

ハヒフヘホ

マミムメモ

ヤユヨ

ラリルレロ

ワヲ (ン)

💬 카타카나 외래어 예시

  • 🍔 コンピュータ = 컴퓨터
  • 🌐 インターネット = 인터넷
  • 📺 テレビ = 텔레비전
  • 📱 スマートフォン = 스마트폰

💡 영어 단어를 보면 카타카나 발음을 유추할 수 있어요! 예: "computer" → "コンピュータ"

⚠️ 한국인이 자주 틀리는 발음 5가지

1️⃣ ら행의 발음

❌ 한국어: "라" (혀를 말아서) / ✅ 일본어: ら (혀를 튕겨서)

2️⃣ し의 발음

❌ 한국어: "시" / ✅ 일본어: し = "쉬"

3️⃣ つ의 발음

❌ 한국어: "츠" (혀를 굴려) / ✅ 일본어: つ = "츠" (혀를 굴리지 않음)

4️⃣ う와 ぅ 구분

う = "우" (일반) / ぅ = "우" (작은 우, 거의 발음 안 함)

5️⃣ へ의 발음

발음은 "에"이지만, 방향을 나타내는 조사로도 사용됩니다.

📌 탁음 (だくおん)

기본 글자에 점(゛)을 붙이면 탁음입니다.

か→が | き→ぎ | く→ぐ | け→げ | こ→ご

🔊 중요한 탁음:

  • じ = "지" / ぎ = "기" / ぐ = "구"
  • ぜ = "제" / ど = "도"

📌 반탁음 · 요음

반탁음 (はんだくおん): は행에만 있습니다. 점(゜)을 붙입니다.

は→ぱ | ひ→ぴ | ふ→ぷ | へ→ぺ | ほ→ぽ

요음 (ようおん): 작은 야행 글자를 결합합니다.

  • きゃ きゅ きょ = "캬, 큐, 쿄"
  • しゃ しゅ しょ = "샤, 슈, 쇼"
  • ちゃ ちゅ ちょ = "차, 추, 초"

📌 장음 (ちょうおん)

음을 길게 늘리는 것입니다.

히라가나에서:

  • あ + あ = ああ → "아아"
  • い + い = いい → "이이"
  • う + う = うう → "우우"
  • え + え = ええ → "에에"
  • お + う = おう → "오오"

🎤 장음 예시:

  • スーパー = "수우파" (슈퍼)
  • コーヒー = "코오히이" (커피)
  • ケーキ = "케이키" (케이크)

📌 촉음 (そくおん)

작은 つ를 사용해 음을 짧게 끊습니다.

🎤 촉음 예시:

  • かった = "깠다"
  • きって = "낏떼" (우표)
  • がっこう = "깍꼬오" (학교)
  • さっき = "쌧끼" (방금)

💡 팁: 한국어 쌍자음처럼 앞 글자를 짧게 끊었다가 다음 글자를 발음합니다!

📌 의성어 · 의태어

의성어 (소리): 실제 소리를 나타냅니다

  • 🔊 ぱたぱた = "팍팍" (발소리)
  • 🔊 わんわん = "멍멍" (개 짖는 소리)

의태어 (모습): 상태나 모양을 나타냅니다

  • ✨ ふわふわ = "폭신폭신" (부드러움)
  • ✨ ぴかぴか = "반짝반짝" (빛남)
  • ✨ つるつる = "매끈매끈" (매끄러움)
  • ✨ ほかほか = "따뜻따뜻" (따스함)

💡 의성어와 의태어는 일본어의 특색 있는 표현입니다. 자연스러운 표현을 위해 자주 접해보세요!

조사 기초: は/を/に/が

일본어 문장을 만드는 가장 중요한 것, 바로 "조사"입니다. 조사는 문장의 각 단어가 어떤 역할을 하는지 알려주는 신호탄이에요. 프롤로그 2강에서 일본어의 핵심인 조사 4가지를 완벽히 마스터해보세요!

💡 팁: 조사를 제대로 이해하면, 일본어 문장 구조가 한눈에 보입니다!
