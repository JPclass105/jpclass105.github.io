✅ 누가, 뭐, 어디에서, 언제, 왜, 어떻게 의문사 6가지를 완벽히 구분할 수 있습니다

✅ 한국어 의문사와 일본어 의문사의 사용 방식 차이를 이해합니다

🎯 의문사(疑問詞)란 뭐예요?

의문사는 "뭔가 모르는 부분을 물어보는 단어"입니다. 한국어의 "누가?", "뭐?", "어디?", "언제?", "왜?", "어떻게?"처럼 말이에요.

의문사의 역할:

• 주어를 물어봄 (누가?) → 誰 (だれ/たれ)

• 목적어를 물어봄 (뭐?) → 何 (なに/なん)

• 장소를 물어봄 (어디?) → どこ

• 시간을 물어봄 (언제?) → いつ

• 원인을 물어봄 (왜?) → なぜ

• 방법을 물어봄 (어떻게?) → どう

💡 팁: 의문사 + 조사 + ます형 = 완벽한 의문문! 문장 끝에 か를 붙이는 것 잊지 마세요!

1️⃣ 누가? / 뭐?

🔤 誰 (だれ / たれ)

誰ですか？

(누구예요?)

誰が来ましたか？

(누가 왔어요?)

💡 발음 팁: "だれ"와 "たれ" 모두 사용되지만, 일상에서는 "だれ"가 더 많이 쓰입니다.

🔤 何 (なに / なん)

これは何ですか？

(이것은 뭐예요?)

何を食べますか？

(뭘 먹어요?)

⚠️ 주의: "何" 다음에 조사가 바로 올 때, "なに" 대신 "なん"으로 발음하기도 합니다.

예) 何の本ですか？ (뭐라는 책이예요?) → "なんの" 발음

2️⃣ 어디에? / 언제?

🔤 どこ (어디)

どこにいますか？

(어디에 있어요?)

どこで勉強しますか？

(어디에서 공부해요?)

💡 팁: "どこに" (어디에) vs "どこで" (어디에서) 차이를 기억하세요!

• に = 목적지 (어디로/어디에) → 존재 / 이동

• で = 장소 (어디에서) → 행동의 장소

🔤 いつ (언제)

いつですか？

(언제예요?)

毎日いつ学校に行きますか？

(매일 언제 학교에 가요?)

💡 발음: いつ는 "이쯔"라고 발음합니다. 한국인들이 "언제"라고 하면서 "이-언-제"처럼 길게 끌기 쉬우니 주의하세요!

3️⃣ 왜? / 어떻게?

🔤 なぜ (왜)

なぜですか？

(왜예요?)

なぜ来ませんでしたか？

(왜 안 왔어요?)

📌 원인 vs 이유: "なぜ"는 원인을 물을 때, "どうして"는 조금 더 부드러운 이유를 물을 때 사용합니다. (나중에 배우기)

🔤 どう (어떻게)

どうですか？

(어떻게요? / 어떠세요?)

毎日どう過ごしていますか？

(매일 어떻게 보내세요?)

💡 팁: "どう"는 "방법"뿐 아니라 "상태/느낌"도 물어봅니다. 따라서 매우 자주 쓰이는 의문사입니다!

📋 의문사별 기본 패턴

의문사 일본어 예문 번역
누가? 誰 誰ですか？ 누구예요?
뭐? 何 何ですか？ 뭐예요?
어디? どこ どこですか？ 어디예요?
언제? いつ いつですか？ 언제예요?
왜? なぜ なぜですか？ 왜예요?
어떻게? どう どうですか？ 어떻게예요?

🎯 기본 공식: [의문사] + 조사 + 명사/동사 + ですか = 의문문

💬 상황별 실전 예문 (1)

📍 교실 상황

田中さんはどこにいますか？

(다나카 씨는 어디에 있어요?)

これは何ですか？

(이것은 뭐예요?)

先生はいつ来ますか？

(선생님은 언제 오세요?)

🏪 쇼핑 상황

これはいくらですか？

(이건 얼마예요?)

どこに置いていますか？

(어디에 놓아두세요?)

💡 팁: "いくらですか?" (얼마?)는 새로운 의문사입니다! 나중에 자세히 배울 거예요.

💬 상황별 실전 예문 (2)

🍵 카페 상황

いつ来ましたか？

(언제 왔어요?)

何を飲みますか？

(뭘 마셔요?)

🎂 일상 대화

誰と一緒に行きますか？

(누구와 함께 가요?)

どうしてそんなことをするんですか？

(왜 그런 일을 해요?)

📌 패턴 주목: 의문사 + 조사 + 명사 + 동사 + ますか?

혼자 자신 있게 문장을 만들어보세요!

⚠️ 한국인이 자주하는 실수 5가지

❌ 실수 1: 의문사를 문장 끝에 넣기

잘못: 学校に行きます、どこに？(X)

올바름: どこに学校に行きますか？(O)

→ 일본어는 의문사가 문장 앞에 와야 합니다!

❌ 실수 2: なぜ와 どうして 구분 못 하기

이 단계에서는 둘 다 "왜?"의 의미인데, 미묘한 차이가 있습니다.

→ 지금은 なぜ부터 확실히 익히세요!

❌ 실수 3: どこに vs どこで 헷갈리기

잘못: どこにご飯を食べますか？

올바름: どこで御飯を食べますか？

→ 행동의 장소는 で, 목적지는 に 기억하세요!

❌ 실수 4: 何を뒤에 바로 명사 붙이기

잘못: 何を好きですか？(X)

올바름: 何が好きですか？(O)

→ 좋아하다는 を가 아니라 が와 함께 써요!

❌ 실수 5: いつ발음 길게 끌기

한국인: "이-언-제" (길게)

올바른 발음: "い-つ" (짧고 빠르게)

→ 음절 단위로 끊어서 발음하세요!

📚 6강 필수 정리: 의문사 완벽 마스터

의문사 6가지 최종 정리

誰 (だれ) - 누가?

何 (なに/なん) - 뭐?

どこ - 어디?

いつ - 언제?

なぜ - 왜?

どう - 어떻게?

의문문 만드는 공식

[의문사] + 조사 + 명사/동사 + ですか?

예: どこ + に + 行き + ますか？ = どこに行きますか？

💡 최고의 팁: 각 의문사를 그냥 외우지 말고, "누가를 물어본다"는 개념으로 이해하세요. 그러면 자동으로 입에서 나옵니다!

✏️ 연습문제

문제 1: 다음 한국어를 일본어 의문문으로 만들어보세요.

→ "누구예요?"

문제 2: 빈 칸에 올바른 의문사를 넣으세요.

＿＿を食べますか？(뭘 먹어요?)

문제 3: 다음 문장을 의문문으로 바꾸세요.

田中さんは毎日学校に行きます。

→ 田中さんは毎日＿＿行きますか？

문제 4: "어디에서 공부해요?"를 일본어로 말해보세요.

힌트: どこ + (조사) + 勉強します

문제 5: 다음 중 올바른 문장은?

① なぜ来ました？ (O / X)

② 来ます、なぜ？ (O / X)

📝 정답 및 해설

문제 1: 誰ですか？

해설: 누가를 나타내는 의문사는 "誰(だれ)"입니다. 문장 끝에 か를 붙여 의문문을 만들어요.

문제 2: 何

해설: "뭘 먹다"는 "何を食べます"입니다. を 앞에 오는 의문사는 "何"입니다.

문제 3: どこに / いつ (또는 どこで)

해설: "어디에 간다"는 "どこに行きます"입니다. 행동 장소를 물을 때는 "どこで"를 써도 괜찮아요.

문제 4: どこで勉強しますか？

해설: "어디에서"는 "どこで"입니다. 행동의 장소를 나타낼 때 で를 씁니다.

문제 5: ① O (O) / ② X (X)

해설: 의문사는 문장 앞에 와야 합니다. ①은 올바른 문장, ②는 한국식 어순이므로 틀렸어요!

대답 표현 완성: はい/いいえ/そうです/ちがいます

이번 강에서는 의문사를 배웠으니, 이제 의문문에 자연스럽게 대답할 차례입니다! "네(はい)", "아니오(いいえ)"뿐 아니라 "그렇습니다(そうです)", "아닙니다(ちがいます)" 같은 고급 대답 표현을 배워봅시다. 이 4가지만 마스터하면, 대화의 60%는 완성된 거예요!
